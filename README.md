## How to Run Locally
```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
```

## Deployment on Streamlit Cloud
1. Push this folder to GitHub
2. Go to [streamlit.io/cloud](https://streamlit.io/cloud)
3. Connect your repo and deploy!
